# Week 0 - Scratch (CS50 2026)

This project is part of Harvard CS50 (2026) course.

## 🎮 Project: Cosmic Catch

Cosmic Catch is a simple arcade game built using Scratch.  
The player controls a frog to catch falling objects and earn points.

## 🧠 Features

- Player movement using arrow keys
- Falling stars mechanic
- Score system
- Diamond bonus item (+5 score)
- Sound effects for interactions
- Win condition when score reaches a target

## 🏁 Win Condition

The game ends when the player reaches the required score, and a win message is displayed.
